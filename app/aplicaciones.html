Subir Aplicación HTML

Arrastre y suelte sus archivos HTML aquí, o haga clic para seleccionarlos.

💡 Instrucciones para Subir Aplicaciones

  • Los archivos deben tener extensión .html o .htm
  • Asegúrese de que su aplicación sea autocontenida (CSS y JS inline)
  • Para aplicaciones con múltiples archivos, comprímalos en un ZIP
  • En Netlify, puede arrastrar su carpeta de aplicación directamente al panel de deploy
  • Para aplicaciones estáticas simples, use el botón "Seleccionar Archivos" arriba
